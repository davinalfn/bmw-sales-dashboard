Proyek ini bertujuan untuk menganalisis dan memvisualisasikan data penjualan global BMW selama periode 2010 hingga 2024 dengan menggunakan Python untuk tahap analisis data serta Power BI untuk pembuatan dashboard interaktif.
Melalui project ini, saya berusaha memahami bagaimana performa penjualan BMW berubah dari tahun ke tahun, bagaimana kontribusi dari setiap wilayah (region), serta bagaimana tren transisi menuju kendaraan listrik (Electric Vehicle) mempengaruhi hasil penjualan secara keseluruhan.

**Latar Belakang**
Perkembangan industri otomotif dunia sangat dinamis, khususnya pada era transisi menuju kendaraan listrik. Sebagai salah satu produsen otomotif besar, BMW menjadi contoh menarik untuk dianalisis karena memiliki data penjualan yang luas dan bervariasi dari berbagai negara.
Melalui analisis data ini, saya ingin melihat **tren penjualan jangka panjang**, **performa antar wilayah**, serta **kategori kendaraan yang mendominasi pasar**. Hasil dari analisis ini divisualisasikan dalam bentuk **dashboard interaktif** agar lebih mudah dipahami dan digunakan sebagai bahan pengambilan keputusan.

** Tujuan Proyek**
1. Perkembangan penjualan secara global dari tahun ke tahun
2. Performa BMW di tiap region (Asia, Eropa, Amerika, dll)
3. Model yang paling laku dan tren kategori kendaraan (SUV, Sedan, EV)
4. Dampak transisi ke kendaraan listrik (Electric/Hybrid) terhadap penjualan
5. Pola pertumbuhan (YoY Growth) dan proyeksi peluang pasar di masa depan

**Alur Pengerjaan Proyek**

1. Data Cleaning dan Persiapan
Tahap pertama dilakukan di Google Colab dengan Python. Pada tahap ini, data mentah dibersihkan dari nilai kosong (missing values), duplikasi, serta dilakukan standarisasi kolom seperti format tanggal dan kategori kendaraan.
Hasil akhirnya disimpan dalam file `BMW_cleaned_final.csv` yang digunakan untuk analisis lebih lanjut dan visualisasi di Power BI.

2. Exploratory Data Analysis (EDA)
Setelah data bersih, saya melakukan analisis eksploratif menggunakan Matplotlib dan Seaborn untuk menemukan pola dan tren.
Beberapa analisis yang dilakukan meliputi :
* Perkembangan penjualan per tahun.
* Distribusi penjualan berdasarkan wilayah (region).
* Analisis perbandingan antar model kendaraan (SUV, Sedan, EV).
* Korelasi antara tahun dan volume penjualan.
Hasil analisis EDA ini menjadi dasar pembuatan insight di dashboard Power BI.

3. Pembuatan Dashboard Power BI
Tahap terakhir adalah membangun dashboard interaktif yang menampilkan insight utama.
Dashboard ini menampilkan:
* Grafik tren penjualan global (2010â€“2024).
* Peta interaktif penjualan per region.
* Perbandingan penjualan antar model kendaraan.
* Persentase kontribusi penjualan kendaraan listrik.
Selain itu, saya juga menambahkan **fitur interaktif** seperti **region filter (slicer)** agar pengguna bisa memilih dan menganalisis performa masing-masing wilayah dengan mudah.

**Hasil dan Insight**
Berdasarkan hasil analisis dan visualisasi yang dibuat, ditemukan beberapa insight menarik:
* Penjualan BMW menunjukkan tren meningkat sejak 2010 hingga mencapai puncak di tahun 2023.
* **Region Eropa dan Asia** mendominasi total penjualan global.
* **SUV** menjadi kategori kendaraan yang paling populer dalam lima tahun terakhir.
* Transisi ke **kendaraan listrik (EV)** semakin kuat dan menunjukkan pertumbuhan signifikan sejak 2020.
